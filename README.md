# Algorithm-Design

https://github.com/blair-cheng/Cs336

Set as a private repository per Northwestern professor's policy. Please reach out for access (current students excluded).

**Language:** C++ (header-only)  
**Assignments:** 8 · **Lines:** ~400

Competitive-programming-style algorithm problems with strict time limits and no partial credit. Eight problems spanning every major paradigm: binary search, greedy (exchange argument), sweep line, sliding window, 1D DP, 2D DP (stock trading with path reconstruction), and Rabin-Karp double hashing for substring matching. The most technically demanding course in the sequence.
